## 📊 Overview

Interactive investor dashboard for Reliance Industries' new venture "Slate" - a digital platform for Indian SMBs and enterprises.

### Key Features

- **💰 Financial Projections**: 3-year revenue, customer, and margin forecasts
- **📈 Unit Economics**: CAC, LTV, Payback Period calculations
- **🌍 Market Sizing**: TAM/SAM/SOM analysis for Indian market
- **📊 Growth Metrics**: Rule of 40, Magic Number, NRR tracking
- **📥 Export Data**: Download complete Excel reports
